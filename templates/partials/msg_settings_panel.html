The easiest option — download and install the Jarvis companion app directly on your phone. No account or sign-in needed.

DOWNLOAD LINK
  1. On your Android phone, open the Download Link above in a browser and install the APK
    • If prompted, allow installs from unknown sources in your phone's settings
  2. Open the app and paste the Webhook URL and Bearer Token from above
  3. Tap Grant Notification Access, find Jarvis Messages in the list, and enable it
  4. Done — Jarvis will now receive your messages silently in the background

Works with Google Messages (RCS + SMS), Signal, WhatsApp, Telegram, and Messenger.

Use Macrodroid (free on the Play Store) to forward notifications to Jarvis — no APK download needed.

  1. Install Macrodroid from the Google Play Store and open it
  2. Tap + to create a new macro, give it a name like Jarvis Messages
  3. Under Triggers → tap Add Trigger → Notification → Notification Received
    • Select the apps you want to forward (e.g. Google Messages, WhatsApp, Signal)
  4. Under Actions → tap Add Action → Connectivity → HTTP Request
    • URL → paste your Webhook URL from above
    • Method → POST
    • Headers → add: Authorization: Bearer YOUR_TOKEN
    • Content Type → application/json
    • Body → paste exactly: {"sender": "{notification_title}", "text": "{notification_text}"}
  5. Tap OK → Save → make sure the macro is toggled on
  6. Grant Macrodroid notification access when prompted — this is required for it to read your messages

Use the built-in Shortcuts app to forward message notifications to Jarvis — no extra app needed. Works with SMS and iMessage.

  1. Open Shortcuts → tap Automation at the bottom → tap + → New Automation
  2. Scroll down and tap App → select Messages → enable Receives a Notification → tap Next
  3. Tap Add Action and search for Get Contents of URL — add it
  4. Configure the action:
    • URL → paste your Webhook URL from above
    • Method → POST
    • Tap Headers → add key Authorization, value Bearer YOUR_TOKEN (paste your token from above)
    • Request Body → JSON
    • Add field sender → tap the value field → choose Shortcut Input → Notification Title
    • Add field text → tap the value field → choose Shortcut Input → Notification Content
  5. Tap Next → turn off Ask Before Running → tap Done
